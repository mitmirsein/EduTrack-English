# EduTrack English 📊
> **중고등 영어 학원 강사용 학생 관리·평가·상담 통합 시스템 (브랜드/학원명은 설정에서 지정)**

---

## 🌟 개요

**EduTrack English**는 중고등학생 영어를 가르치는 학원 강사, 공부방 운영자, 과외 교사를 위한 **무설치 로컬 단독 실행형 웹 어플리케이션(SPA)**입니다.

복잡한 데이터베이스 서버나 유료 호스팅이 필요 없으며, 모든 데이터는 강사님의 브라우저 내부 스토리지(`LocalStorage`)에 저장됩니다. 초심자 강사님도 파일 하나만 더블클릭하면 즉시 실무에 투입할 수 있도록 설계되었습니다. 학생 관리·성적 추적부터 약점 진단, 반 운영, 교안·단어 해설까지 **수업 사이클 전체**를 한 도구 안에서 다룹니다.

## 🚀 사용자(강사) 환경별 즉시 사용법

> ### ⚠️ 시작 전 꼭 읽어주세요: 데이터는 "실행 방식"마다 따로 저장됩니다
> 입력하신 데이터는 브라우저 내부(LocalStorage)에 저장되는데, 이 저장 공간은 **실행 방식과 브라우저별로 완전히 분리**되어 있습니다.
> - 더블클릭 실행 ↔ `start_edutrack.bat` 실행 ↔ 웹주소 접속 → **서로 데이터가 보이지 않습니다.**
> - Chrome ↔ Edge 브라우저를 바꿔도 → **서로 데이터가 보이지 않습니다.**
>
> 데이터가 사라진 것이 아니라 저장 위치가 다른 것입니다. **아래 방식 중 하나를 정해 계속 그 방식으로만 사용**하시고, 방식을 바꿀 때는 반드시 기존 방식에서 **[📥 데이터 전체 백업]** → 새 방식에서 **[📤 백업 파일 복구]** 절차를 거치세요. 현재 어떤 방식으로 실행 중인지는 대시보드 상단 명판에 표시됩니다.

### 1. [무설치/추천] Windows 11에서 `start_edutrack.bat` 더블클릭
1. 이 저장소를 **Code → Download ZIP**으로 내려받아 압축을 풀거나, `git clone` 합니다.
2. 폴더 안의 `start_edutrack.bat` 파일을 **더블클릭**합니다. (Node.js가 없으면 설치 안내가 표시됩니다 — `winget install OpenJS.NodeJS.LTS` 한 줄이면 됩니다.)
3. 최초 1회 자동 설치(1~2분) 후 브라우저가 **`http://localhost:5175`** 로 자동으로 열립니다. (포트가 5175로 고정되어 있어, 다음에 다시 실행해도 같은 주소·같은 데이터로 열립니다.)
4. 처음 실행 시 Windows SmartScreen 경고("알 수 없는 게시자")가 뜨면 **[추가 정보] → [실행]**을 누르면 됩니다. 검은 콘솔 창을 닫으면 서버가 종료됩니다.

> **🍎 macOS 사용자**: `start_edutrack.bat` 대신 같은 폴더의 **`start_edutrack.command`** 를 더블클릭하세요(동작은 동일, http://localhost:5175). 처음 실행 시 "확인되지 않은 개발자" 경고가 뜨면 파일을 **우클릭 → [열기]**로 한 번 통과시키면 됩니다. (Node.js가 없으면 `brew install node` 또는 nodejs.org 안내가 표시됩니다.)

### 2. [원클릭 호스팅] GitHub Pages를 통한 나만의 온라인 사이트 구축
이 레포지토리를 포크(Fork)하면 모바일·태블릿·다른 컴퓨터에서도 주소만 치면 접속할 수 있는 무료 웹사이트가 자동으로 만들어집니다.
1. 우측 상단의 **Fork** 버튼을 눌러 본인 깃허브 계정으로 가져옵니다.
2. 포크된 저장소에서 **Settings → Pages**로 들어갑니다.
3. **Build and deployment**의 Source를 **`GitHub Actions`**로 선택합니다.
4. 상단 **Actions** 탭에서 배포 워크플로가 완료(초록 체크)되면 `https://본인아이디.github.io/EduTrack-English/` 로 접속합니다.

### 3. [단일 파일] 바탕화면 더블클릭용 HTML 직접 생성 (개발자/관리자용)
아래 빌드 명령으로 생성되는 `dist/index.html` 한 파일만으로도 동작합니다. 단, 대시보드의 평가 자료실 PDF 링크까지 쓰려면 `level_test/`·`appendix_test/pdf/` 폴더를 같은 위치에 함께 복사해야 합니다.

## 🛠️ 핵심 기능

### 1. 종합 대시보드
*   재원생·상담생 수, 누적 상담·성적 데이터를 한눈에 보는 지표 스트립(클릭 시 해당 화면으로 이동).
*   **이탈 조기경보**: 점수가 낮지 않아도 **성적이 하락 추세이거나 급락한 재원생**을 자동으로 찾아 맨 위에 띄웁니다.
*   **재시험 대상**(60점 미만)과 **최근 입력 활동**, 그리고 레벨테스트 PDF 자료실 바로 열기.

### 2. 학생 관리 & 성적 추적
*   **학생 카드**: 학교·학년·**수강 반**·연락처·특이사항을 통합 관리. 레벨테스트 실시 여부도 표시.
*   **성적 누적 입력 & 추이 차트**: 입력한 **모든 성적 유형**별로 점수 추이가 SVG 선형 차트로 그려져 취약 구간을 한눈에 파악합니다.
*   **영역별 약점 진단**: 레벨테스트·성취도 평가에서 마킹한 **오답을 문항 영역별로 누적 분석**해, 어휘/문법/구문/독해 중 어디가 약한지 정량(오답률)으로 보여주고, 그 약점에 맞는 **교안을 추천**합니다.
*   **학부모 소통 타임라인**: 상담지·분기 리포트 발송 이력을 모으고, 마지막 소통 후 오래 지나면 환기합니다.

### 3. 레벨테스트 분석 & 학부모 상담지 빌더
*   **5대 영역 진단**: 어휘(Vocab)·문법(Grammar)·구문(Syntax)·독해(Reading)·영작(Writing) 점수로 실시간 **방사형 레이더 차트**를 렌더링.
*   **1차 진단서 자동 작성**: 설정 탭에 Google Gemini / OpenAI API Key를 등록하면 학원 기준표에 맞춘 분석서 초안을 AI가 작성하고, Key가 없으면 점수대별 **룰셋 템플릿**으로 즉시 생성합니다. 강사가 2차로 정밀 편집할 수 있습니다.

### 4. 반별 운영
*   관리(기준정보)에 등록한 **과정(반)별로 학생 명단·인원·반 평균(최근 성적)·연결 교재**를 한눈에 봅니다. 미배정 학생도 별도로 표시합니다.

### 5. 교안 자료실
*   수업 교안 PDF나 퀴즈 HTML을 **코스·레벨**로 분류해 등록하고, 앱 안에서 열람·인쇄합니다. (파일은 브라우저 내부 IndexedDB에 보관)

### 6. 단어 해설 (어원 기반)
*   단어를 입력하면 **어근을 분해해 1차 뜻이 2차 뜻으로 확장되는 논리**를 학생 눈높이로 풀어주는 해설이 AI로 생성됩니다(어원·의미 확장은 단어마다 달라 API Key 필요). 단어별로 저장·인쇄하고 `.md`로 내려받아 `wordbank/` 폴더에 모을 수 있습니다.

### 7. 관리 (기준정보)
*   **학교·학년·과정(반)·교재/챕터·성적 유형**을 강사가 직접 등록·수정·삭제합니다. 코드 수정 없이 학교나 교재를 추가할 수 있고, 여기 등록한 목록이 학생 등록·상담·성적 입력 화면의 선택 목록(드롭다운)이 됩니다.

### 8. A4 인쇄 & 고해상도 PDF
*   상담지·분기 리포트·종합 분석 리포트·시험지·단어 해설 모두 **EduTrack English** 로고가 박힌 A4 프리뷰로 출력합니다. 브라우저 인쇄에서 **"PDF로 저장"**을 선택하면 학부모 전송용 PDF가 깔끔하게 저장됩니다(내용이 길면 여러 페이지로 자동 분할).

### 9. 안전한 로컬 백업 & 복원
*   주기적으로 **[📥 데이터 전체 백업]**으로 `.json` 파일을 받아 보관하세요. 컴퓨터 교체·포맷 시 **[📤 백업 파일 복구하기]**에 업로드하면 복구됩니다(복구 직전 현재 데이터도 자동 백업).
*   **개인정보 주의**: 백업 파일에는 학생·학부모 연락처가 들어 있습니다. 전송 시 취급에 주의하세요. (API Key·PIN은 백업에 포함되지 않습니다.)

### 10. 실행 잠금 (4자리 PIN)
*   **설정 탭 → [실행 잠금 (4자리 PIN)]**에서 잠금을 켜면 앱을 열 때마다 PIN을 입력해야 데이터 화면이 표시됩니다. 무활동 자동 잠금(5/10/30분)과 사이드바 **[지금 화면 잠그기]**도 제공됩니다.
*   **한계 안내**: 공용 PC에서 어깨너머 열람·잠깐 자리 비움을 막는 보호 장치입니다. 데이터 자체는 암호화되지 않으므로 개발자 도구를 다루는 사람의 접근까지 막지는 못합니다.
*   **PIN을 잊었을 때 (데이터는 안전)**: 잠금 화면에서 `F12` → **Console** 탭에서 `localStorage.removeItem('edutrack_lock')` 입력 후 Enter → 새로고침(F5)하면 잠금이 풀립니다. PIN은 백업에 포함되지 않습니다.

## 💻 개발자용 빌드 가이드

```bash
# 의존성 패키지 설치
npm install

# 로컬 개발 서버 실행 (http://localhost:5175 고정)
npm run dev

# 린트 / 단일 index.html 빌드
npm run lint
npm run build
```

*   **단일 파일 배포**: `vite-plugin-singlefile`로 빌드 시 `dist/index.html` 한 개만 생성되며 모든 CSS·JS가 HTML 내부로 컴파일됩니다.
*   **dev 포트 고정**: `vite.config.js`에서 포트를 **5175로 고정(strictPort)**했습니다. 데이터(LocalStorage)는 origin(포트)별로 분리되므로, 포트가 바뀌면 데이터가 갈라지는 사고를 막기 위함입니다.
*   **데이터 구조**: 학생/성적/상담/분기리포트/기준정보(master)는 `edutrack_*` LocalStorage 키에, 교안·단어 해설 파일은 IndexedDB에 저장됩니다. 시험지 콘텐츠 등 작업 규약은 `AGENTS.md`를 참고하세요.
